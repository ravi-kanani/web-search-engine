Templates and Addons for WPBakery Page Builder (formerly Visual Composer)

Look at our new WPBakery Page Builder addon. Create amazing pages from ready to use template with only single click. There are more than 38+ addons with 5 ready to use templates, just you need to edit it with your own content.

Bundled with super useful WPBakery Page Builder (formerly Visual Composer) elements with bunch of options to achieve any design with all the power of WPBakery Page Builder for free.

You can build amazing professional layouts for your site with tons of powerfull shortcodes. 38+ addons with one plugin. Works with latest WPBakery Page Builder v5.2.x

Installation

Installing this plugin as regular wordpress plugin. First you need to install WPBakery Page Builder plugin then install Essential WPBakery Page Builder.

This addon plugin works like other page builder addon plugin. Go to new page then click on WPBakery Page builder then click on Add Element you should see VC Addons menu as Name “VC Addons” on Menu Top, click on it you will see shortcode item list.
