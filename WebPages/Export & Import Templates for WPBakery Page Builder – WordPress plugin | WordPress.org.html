Export & Import Templates for WPBakery Page Builder

Description

Export & Import templates for WPBakery Page Builder (Saved Templates/My Templates) in few clicks.

This plugin is addon for WPBakery Page Builder.

Features

  • Import/Export templates in JSON format – take your custom templates.
  • Single Export – export only those templates you wish.
  • Multi Choose – export multi choose templates you wish
    (Premium).
  • Export All – Allow export all templates with a click
    (Premium).
  • Multisite (Network) – export & import templates from one place.
    (Premium).

Installation

  1. Upload the plugin files to the “/wp-content/plugins/vc-templates-import-export/” directory, or install the plugin through the WordPress plugins screen directly.
  2. Activate the plugin through the “Plugins” screen in WordPress.
  3. WPBakery Page Builder -> Templates Import & Export screen to use the plugin.
