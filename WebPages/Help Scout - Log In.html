Looking for a better way to talk with customers? Try Help Scout

Help Scout requires JavaScript

Follow these steps to enable it in your browser

Google Chrome (Windows)

  1. Click the wrench icon on the top right of the browser window and select "Options".
  2. Click the "Under the Hood" link in the sidebar.
  3. Click the "Content Settings ..." button on the top of the page.
  4. In the Javascript section of the window, select "Allow all sites to run JavaScript".

Google Chrome (Mac)

  1. In your browser's Chrome menu, click Preferences.
  2. Click the "Under the Hood" link in the sidebar.
  3. Click the "Content Settings ..." button on the top of the page.
  4. In the Javascript section of the window, select "Allow all sites to run JavaScript".

Firefox 3+ (Windows)

  1. In your browser's Tools menu, click Options.
  2. Click the Content tab.
  3. Check the box next to "Enable Javascript"
  4. Click OK.

Firefox 3+ (Mac)

  1. In your browser's Firefox menu, click Preferences.
  2. Click the Content tab.
  3. Check the box next to "Enable Javascript"

Microsoft Edge

  1. On the web browser menu, click Tools or the "Tools" icon (which looks like a gear), and select Internet Options.
  2. When the "Internet Options" window opens, select the Security tab.
  3. On the "Security" tab, make sure the Internet zone is selected, and then click on the "Custom level..." button.
  4. In the Security Settings — Internet Zone dialog box, click Enable for Active Scripting in the Scripting section.
  5. When the "Warning!" window opens and asks, "Are you sure you want to change the settings for this zone?" select Yes.
  6. Click OK at the bottom of the Internet Options window to close the dialog.
  7. Refresh this page.

Internet Explorer 11+ is required
