Private DNS servers

Private DNS servers are fully white-label DNS servers. When you get a Private DNS server, it will be linked with our network and web interface. The Server will be managed and supported by our system administrators and you will be able to manage all your domains via our web interface. Every Private DNS server has included:

All Premium features - TTL management, Secondary DNS, Cloud domains, Dynamic DNS, SOA Settings and Hourly statistics

Unlimited DNS zones. You can host as many DNS zones that your server can handle. Will be monitored 24x7 and our team will contact you, if any limit of the server is reached. We will provide detailed information (graphics and logs) from our monitoring.

Unlimited DNS records. You can host as many DNS records that your server can handle. It will be monitored 24x7 and our team will contact you, if any limit of the server is reached. If you need, we will provide you detailed information (graphics and logs) from our monitoring.

Only locations you need. It is not required to have 10 or more Private DNS servers, you can purchase Private DNS servers only in locations around your customers.

You can purchase Private DNS server with resources that fit your needs. It is not required to pay for support, system administration and hardware that you don't need.

All Private DNS servers are managed and supported by our team. All features and functionality of our system will be deployed to your private server.

Private DNS servers have a dedicated IP address and PTR record. They can be used for white label DNS service for reselling.

Our HTTP API can be used for a full integration with your system

One business day delivery time
