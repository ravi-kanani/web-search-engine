opacity is, and turns out visibility is too. this combination is pretty useful together, as once an element has visibility: hidden; it doesn't interfere with events like clicks/taps. this combo doesn't take the element out of page flow though, which can be useful or not useful. the position property might be useful in those circumstances. see - so much to think about! our learning partner need some front-end development training?
