activello theme is a fairly simple blog theme and is very easy to install and setup but here are some tips and videos that will make your life much easier. for more information abut this theme please see theme demo and its landing page. here are things that you will learn on this guide: how to install activello theme how to setup front page and slider how to setup main menu how to setup sidebar widgets how to use instagram widget how to activate social media icons how to upload logo 1. how to install activello theme here is a short video that will show you have to install activello theme on your wordpress website assuming that you already have hosting and wordpress installed on it. if now you can follow this detailed guide on how to create your own wordpress website. 2. how to setup frontpage and activate slider on this video you will learn how to setup your website frontpage just like on activello theme demo. this also covers features slider setup. featured slider for this theme is based on posts and category to which you have assigned those posts. if you don’t have any posts then slider won’t work because there won’t be any content to display. this theme doesn’t allow to use pages on slider and you have to use posts. this is a blog theme after all. 3. how to setup main menu activello theme depends on default wordpress custom menu functionality therefore setup works the same way as for any other wordpress theme. for more information you can watch this short video which will get you started: 4. how to setup sidebar widgets this theme comes with 3 different widgets for categories, recent posts, social icons and it depends on some 3rd party widgets such as ones that comes with jetpack. we use jetpack widget for subscriptions and about section. for more information please watch this video: 5. how to use instagram widget there are many ways to add instagram feed on your website and you can use some of free instagram plugins that are available on wordpress plugin repository or use snapwidget like we are using for activello theme demo. for more information please watch video below. 6. how to activate social media icons activello theme uses wordpress custom menu in combination with custom widget made specially for this theme to handle social links. one option is to have social links in the footer of your website and the second options is to have links inside sidebar using “activello social widget”. we have covered both of these options in this short video. 7. how to upload logo there are 4 different logo, site description and site title options for this theme: logo only logo + tagline title only title + tagline logo, site title, and site description can be changed via appearance >> customize >> site identity. for more information please see this
