18. dez 2019 fast jeder computernutzer hat es schon mal selbst erlebt: datenverlust. ohne permanentem backup kann frust entstehen, wenn die lieb gewonnenen digitalen schätze ins nirwana verschwinden und nicht mehr zu retten sind. mit dem passenden tool lässt sich das verhindern. datensicherung ist weder kompliziert noch teuer. doch ein backup zu erstellen, dazu noch seine daten zu verschlüsseln und diese maßnahme regelmäßig zu wiederholen, machen nur die wenigsten computernutzer. zwar gibt es seit windows 8 eine fortwährende sicherung von daten, aber mit geeigneten tools lässt sich ein datenverlust noch umfassender vorbeugen und an die eigenen wünsche anpassen. daten-security für gehobene ansprüche dabei lohnt sich ein blick auf ein tool, das neben den typischen windows-bordmitteln weitreichendere optionen bietet, mit der du deine wichtigsten inhalte sicher und zuverlässig vor verlust schützen kannst. easeus todo backup 12.0 ist eine solche software, mit der du ein vollständiges backup deiner partition erstellen kannst. im notfall lässt sich dein system so in windeseile auf den zuletzt gesicherten stand zurückversetzen. mountest du das angelegte image, kannst du auch einzelne dateien oder systemelemente wieder zurückholen. das ist natürlich für all diejenigen interessant, die in einem bestimmten ordner (zum beispiel in einer medienbibliothek oder unter favoriten) ihre wichtigen daten sammeln. das tool sichert darüber hinaus auf wunsch auch deine dateien automatisch ab. ändern sich diese, kannst du darüber einen automatisch erstellten hinweis per e-mail erhalten. das backup lädst du entweder in einer cloud deiner wahl hoch oder sicherst es auf einer externen festplatte oder auf einer cd beziehungsweise dvd. eine boot-cd kannst du ebenso für den notfall erstellen. automatische backups anlegen nach plan für die erstellung einer sicherungskopie deines windows-betriebssystems reicht bei easeus todo backup ein klick auf „system backup“. dann sucht das tool die richtige partition aus. das „smart-backup“ empfiehlt dir, regelmäßig wichtige dokumente zu schützen. wählst du diese option aus, sichert das tool die veränderten daten in festen zeitspannen. so kannst du zum beispiel bestimmen, dass wöchentlich am montag oder täglich um 23 uhr deine daten gespeichert werden. wählen kannst du dabei auch zwischen „vollsicherung“, „inkrementellem“ oder „differenziellem backup“. anschließend kannst du die daten komprimieren und verschlüsseln lassen. legst du mehr als eine backup-version an, löscht easus todo backup 12.0 ältere versionen entsprechend des alters. ebenso kannst du mehrere images zusammenführen beziehungsweise mergen. easeus todo backup: vielseitiges backup-tool auch für unternehmen neben der freeware-version für privatnutzer bietet easeus eine kostenpflichtige variante an, die auch für unternehmen interessant ist. damit lässt sich zum einen die wiederherstellung auch von anderen systemen herbeiführen, zum anderen ist ein spezielles outlook-backup möglich. nützlich ist dabei auch die option, images oder backup-archive zu mounten. diese lassen sich wie ganz normale dateiordner öffnen, so dass du deine dateien selektiv zurückspielen lassen kannst. aktuell bietet easeus sein programm todo backup in version 12.0 in einer 1-jahreslizenz für knapp 16 euro an. jetzt mehr erfahren teile diesen beitrag! unsere empfehlungen
