read the guide what is a website builder? website builder is a tool built to help you create a website without any manual code editing and it provides the all-in-one solution to build a website or blog. unlike cms platforms that have to be installed and configured on your web hosting account, website builders allow you to create websites, blogs and e-commerce stores inside their own platform. your website is designed and built to be hosted only on this platform as outlined in web builders’ terms of use. so it’s important to choose an option you can live with in the future. the downside of that setup is that once you select your builder and create a website, you cannot take what you created and move it somewhere else. types of website builders there are two types of website builders available for you to choose: offline and online offline website builders offline website builders allow you to make your website using their downloadable software and save the files on your local computer. once you are done with your website, you just need to upload all the files to a web host (e.g. rapidweaver). good thing about offline website builders is that you are building your website on your computer and you can work on it even if you are offline. but keep in mind that you need to have at least some technical skills in order to upload those files to a web host down the road. and don’t forget that you will also need to purchase a web hosting account. online website builders online website builders are web-based and you can create your website online using your internet connection and you don’t need to download software on your computer like you would when using offline builders (e.g. wix). good thing about online website builders is that they already come with web hosting services and you don’t need to purchase it separately. online website builders require little or no coding experience at all due to their user-friendly interface. our recommended web builder wix create your own beautiful website by choosing from a huge selection of designer-made templates. use their easy drag n’ drop option to change, customize and add anything to your site. learn more
