The Best Free WordPress Font Plugins To Change Fonts With Ease 2020

  • March 27, 2020
  • Aigars Silkalns
  • Plugins
  • 2 Comments

A major and important part of any web design is typography. The type of fonts that you use on your website reflects the nature of your website and can help to reinforce your brand in the visitors’ minds. For instance, you may want light and attractive fonts on a website that’s meant for kids. Or, if you need a website for your rock band, some outlandish or funky fonts can help. For a lawyer’s website, you may want sturdy and solid fonts.

The type of fonts that you use is mostly determined by your theme. Most premium themes come with inbuilt access to Google fonts and allow you to tweak the theme files to alter the fonts. Developers can also add external fonts to WordPress themes. But if you’re not familiar with code, and don’t want to mess with the theme files, you can simply look to the many WordPress font plugins that can help to add and modify fonts on your website.

If you’re using video elements, interactive infographics or dynamic media environments, you may want to try free flowing hipster fonts. But to generally add fonts to your website and to modify them as you wish, you can use WordPress font plugins. Take care to choose plugins based on user interface and ability to upload custom fonts.

Free WordPress Font Plugins

The WordPress plugin repository has a number of free font plugins that help you to add Google fonts to your website. Many of them allow you to add custom fonts as well, and allow you to work from a user friendly interface.

Easy Google Fonts

Easy Google Fonts is compatible with any theme and is a convenient plugin to add Google fonts to your WordPress theme, without using code. Since the plugin integrates with the WordPress Customizer, you can preview the fonts in real time. By helping you make changes to font controls in your theme and make it available in the admin area, it pretty much gives you full control over the typography of your theme.

And with some basic knowledge of CSS selectors, you can add custom font rules. You get to choose from over 600+ Google fonts and font variants. The Google font list is updated automatically with the latest fonts. The plugin is a useful tool for theme authors who want to add Google fonts to their themes.

More info / Download

Use Any Font

Use Any Font allows you to use any custom font you want on your website without having to use code. It does not restrict you to the fonts of any particular service, nor does it involve storing your fonts on a remote server. And so, you’re not dependent on any third party server’s uptime. All the fonts are stored on your own server, and the conversion happens on the plugin server, after which it is returned to the site.

The plugin supports all major browsers and font conversion happens within the plugin interface. The uploaded fonts can be used directly from the WordPress Editor. Uploaded custom fonts can be used directly from the theme options panel of many premium themes. Multiple custom fonts are allowed (Premium), and the plugin is fast and SEO friendly. An API key is necessary to access the plugin’s server for font conversion. You can access one font set free, but if you need more than one font set, you’ll need to pay for the API keys. Custom font sizes upto 15MB is allowed.

More info / Download

Fonts

Fonts adds options to your visual editor. This makes it possible for you to make changes to the fonts and accept them even as you are working on your post or page. Two options are added in a drop down format – one option for font sizes and the other option for font styles. Working with the plugin is similar to working with any word processor.

To be able to add you own custom fonts, you’ll need to upgrade to the premium version of the plugin.

More info / Download

TinyMCE Advanced

The Visual Editor in WordPress makes it easy for you to work within WordPress by allowing you to quickly call on the options in the toolbar. But not all the options can be accessed from the toolbar. The TinyMCE Advanced plugin lets you add, delete and edit buttons in the visual editor toolbar.

The TinyMCE plugin does not add fonts to your WordPress, but it allows you to add up to four rows of buttons to the toolbar. You can add buttons to adjust font sizes and font family and to format your pages and posts.

More info / Download

Zeno Font Resizer

If the Zeno Font Resizer is active on your website, you can give your readers the option to increase or decrease the font size as they wish. The readers see the same sizes when they revisit the site, as the settings are saved in a cookie. The plugin uses JavaScript and jQuery to set the size of the fonts.

The plugin adds a font resizer widget to your sidebar. Or you can add the template code on any page or post. The content which can be resized can be defined in the admin page. The default option resizes all the content on your site. Since plugin uses jQuery, users can adjust font size without having to reload the page.

More info / Download

Custom Fonts


If you would like to spice up the experience of your existing website with custom fonts, well, you better get your hands on Custom Fonts. This free WordPress font plugin is super easy to use, yet it can create a strong and rewarding impact on your website. First, you install the tool. Second, you activate it in your WordPress admin dashboard. And then, well, you are ready to upload font files of all sorts of different files, like woff2, ttf, svg and eot.

Bear in mind, Custom Fonts works flawlessly with Astra and Beaver Builder themes, as well as Beaver Builder plugin and Elementor drag and drop page builder. You will find all your new fonts in the settings sections where you can pick them from a drop down menu. Moreover, if you are not using any of the mentioned themes and plugins, you can also write a custom CSS to add new fonts.

More info / Download

Styleguide


Styleguide styles your WordPress theme’s fonts and colors in the live customize section. Before you go live, play around with different options and find the right look that will best represent your project online. With Styleguide, you can effortlessly add a distinct touch to your online presence and make it more memorable. Besides, anyone can make changes and improvements with Styleguide. In other words, there is no need for you to hire a designer nor a coder to do it for you.

Moreover, Styleguide supports all default WordPress themes and, with future updates, you can expect the tool to work with loads of other web design as well. Also, Styleguide supports a whole bunch of different character sets that you can use to fine-tune your page. From Cyrillic and Devanagari to Greek, Hebrew and Latin, Styleguide works with all and then some.

More info / Download

OMGF


OMGF stands for Optimize My Google Fonts, allowing to host Google Fonts locally. With OMGF, you can leverage browser cache and minimize DNS requests. OMGF optimizes your website or blog further, helping you boost the page load speed even more. For your information, the entire configuration of OMGF is pretty damn simple. However, if you are not really sure what you are doing, talk to your web developer first and go from there.

Moreover, OMGF detects which Google Fonts your theme uses, downloads them, generates a spreadsheet and automatically adds them to your header. You can also change the caching patch, preload the entire stylesheet or just fonts above the fold and automatically generate local source for fonts. There is always room for improvement, and maybe you have not yet thought of your fonts. Now you know.

More info / Download

Google Fonts Typography


If you would like to enrich your website or blog with different fonts, Google Fonts Typography will help you do that. It allows you to pick any from over nine hundred Google Fonts and integrate it into any WordPress theme that you use. Also, you can easily configure one font for the heading and one for the content, no need to stick to the same one throughout the entire website. What’s cool about Google Fonts Typography is the live preview that offers you to check first how the font looks on your page before you go live.

Some of the features of Google Fonts Typography are one click updates, SEO-friendliness, hosting Google Fonts locally and complete support for different languages. In the typography settings, you can also configure font size, color, line heights and weights.

More info / Download

Dehkadeh Fonts


When you are translating your website, you can utilize the power of Dehkadeh Fonts and introduce Persian language. This free WordPress font plugin has all the necessary to set Persian fonts in the WordPress Customizer. What’s cool, with the plugin, you can also customize the WordPress’ admin dashboard. In short, Dehkadeh Fonts is a full-blown free WordPress plugin for Persian users.

In the bundle, Dehkadeh Fonts has 120 available fonts, but you can easily upload custom fonts as well. The process of installation and activation of the plugin is exactly the same as of any other plugin you ever installed. Once live, you can start benefitting from its features right away. In addition to that, if you need any extra support and assistance, you can always hit up plugin’s author.

More info / Download

Arabic Webfonts

Arabic Webfonts is another quality plugin to add Arabic fonts to your WordPress website. It adds customs font controls for your theme and controls the typography in real time. It integrates with your WordPress Customizer, so you can view the font changes even as you make them. It’s compatible with most standard WordPress themes.

Moreover, it allows you to create custom font controls in the admin area and make it available in the customizer area for use, without using code. With the plugin active on your site, you can control font size, type and style for each and every section in real time.

More info / Download

WP Google Fonts

WP Google Fonts makes it easier to add fonts from the Google font library to your website. It helps you add the site specific Google code to your WordPress. From your WordPress settings, it also allows you to assign the fonts to specific CSS elements. Or, you can even assign the fonts working from your own stylesheet.

The plugin is regularly updated to take advantage of new additions to the Google font library.

More info / Download

Premium WordPress Font Plugins

This list of WordPress font plugins includes a single premium plugin. This plugin is devoted entirely to management of fonts, unlike many other premium plugins which include font management as one of their features.

FontPress

FontPress is a powerful plugin that helps you totally manage all your fonts. It offers a wide range of fonts letting you choose from Cufons, Google and Adobe Edge web fonts, Adobe Typekits, @font-Face fonts or simple CSS fonts. Not only that, it also supports Early Access Fonts, Google’s new fonts for non Latin languages.

To use the plugin, select the fonts you want on your website and create global rules for their use. A number of parameters help to adjust font height and width, weight, color, background color, outline and shadow effect. Transform options for capitalization, font styles and decoration options are available. And if you want to prevent others from copying content, you can choose to turn text into an unselectable entity. Using typography shortcode, you can also create custom text blocks.

The font preview feature lets you view the fonts in all combinations of bold and italics. It includes CSS3 responsive units for font size and line height.

More info / Purchase

Self-Hosted Google Fonts Pro


Self-Hosted Google Fonts Pro is a WordPress plugin that will help boost the performance of your website. When utilizing this plugin, there is no need for CSS to edit, fonts to download or upload and no need to perform any changes and configurations to your themes or plugins. The integration is quick and easy, allowing you to refine your web space in a snap of a finger.

Some of the features of Self-Hosted Google Fonts Pro are automatic processing and conversion, Unicode-Range and font-display behavior to name a few. As for the latter, browsers have invisible text and your font only appear when downloaded. That’s when Self-Hosted Google Fonts Pro comes into play. It has a unique feature that uses a fallback font immediately. To elevate the experience, you might consider taking a peek at Self-Hosted Google Fonts Pro.

More info / Purchase

In Conclusion

Fonts help to increase user engagement on your website and enhances user experience. They impart character to your site and reinforce brand value. The plugins in this list will help you add, customize and manage fonts on your website. And before you select a particular font, take a minute to check that it’s free for use. It could save you some legal trouble later.
