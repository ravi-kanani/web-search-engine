Ultimate Carousel For WPBakery Page Builder

By Nasir
  • Details
  • Reviews

  • Support
  • Development

Description

With Ultimate Carousel, you can create carousels for any kind of your content. We’ve integrated into WPBakery Page Builder to bring you a very flexible, very lightweight and very intuitive carousel creation system. Carousel Anything is responsive and touch enabled. In touch devices, you can drag and swipe the carousel to move it.
WPBakery Page Builder (formerly Visual Composer) plugin must be installed and activated to use this plugin. After you activate the required plugins, the elements should be available for use in WPBakery Page Builder.

Features

  • Turn any content into a carousel (Images, Image with text,)
  • Post Carousel and Post Grid design
  • Slider With Content
  • Image Slider
  • Video showcases,
  • Testimonials
  • Team Members
  • Dots, arrows or none

Installation

  1. Go to plugins in your dashboard and select ‘add new’.
  2. Search for ‘Ultimate Carousel For WPBakery Page Builder’ and install it.
  3. Go To pages and create design with help of viusal composer page builder.
  4. Fill some additional informations.
  5. Now visit your site
